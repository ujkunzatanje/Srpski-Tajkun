# Map selection update

The start screen now shows two visual map previews directly below the player-name field:

1. **Opštine Beograda** — 13×13 board with stadiums.
2. **Gradovi Srbije** — original 11×11 board.

No map is preselected. The **Napravi sobu** button remains disabled until the host clicks one of the previews. Joining an existing room by code or room link does not require choosing a map because the room already has one assigned.


## v1.4
- Map previews reduced to about half their previous desktop size.
- Rent-free trade conditions now support 1–5 circles.
- A circle is consumed when the beneficiary reaches/passes START.
- Active conditions are shown under each affected player's card.
