# Srpski Tajkun multi-map v1.5

- The whole desktop sidebar (Igrači, Razmene and Dnevnik igre) now scrolls independently from the board.
- A narrow visual divider separates the board from the sidebar.
- House, hotel and stadium upgrade buy/sell controls lock immediately after a click and unlock only after the server acknowledges the action or the request times out.
- The server returns an acknowledgement for every building action, including validation errors.
- The changes are shared by both Gradovi Srbije and Opštine Beograda maps.
